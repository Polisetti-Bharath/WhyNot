# WhyNot - Career Intelligence Platform

WhyNot is a modern, AI-powered career intelligence and placement platform designed to help students and professionals optimize their job search and career trajectories.

## Key Features

- **AI Resume Analyzer:** Leverages Google Gemini to provide actionable feedback on resume strength and ATS optimization.
- **Career Path Simulator:** Generates personalized, milestone-driven career roadmaps based on user profiles.
- **External Jobs Board:** Aggregates real, verified software and tech job listings dynamically from multiple live APIs (JSearch, Remotive).
- **Modern UI:** Built with React, Tailwind CSS, and Framer Motion for a sleek, responsive, and accessible dark/light mode user experience.

## Tech Stack

- **Frontend:** React (Vite), TypeScript, Tailwind CSS, Framer Motion
- **Backend / Database:** Supabase
- **AI / ML Integrations:** Google Generative AI (Gemini 1.5 Flash), HuggingFace
